PART 7 · Higher-Degree Inequalities · 🟠 Trigger
จุดวิกฤตและตาราง + − +
🎯
ขวาสุดบวกก่อนเสมอ แล้วสลับไปซ้ายทีละช่อง (ถ้าไม่ซ้ำ)
👀 เมื่อเจอโจทย์แบบนี้
  • \( (x-1)(x+2)(x-3) > 0 \)
  • \( x^2-5x+6 \le 0 \)
✅ วิธีคิด
  1. จัดให้ฝั่งขวาเป็น 0 ก่อน
  2. Factor ให้สุด หาจุดวิกฤตทั้งหมด
  3. เรียงจุดวิกฤตบนเส้นจำนวนจากน้อยไปมาก
  4. ช่องขวาสุด = บวก แล้วสลับทีละช่อง (สำหรับกำลังคี่)
  5. ดูเครื่องหมายของอสมการ → เลือกช่วงที่ถูกต้อง
❌ จุดที่พลาดบ่อย
  • ลืมย้ายข้างให้ฝั่งขวาเป็น 0 ก่อน
  • ลืมเรียงจุดวิกฤตจากน้อยไปมากบนเส้น
  • สัมประสิทธิ์นำหน้าเป็นลบแต่ไม่ดึงออกก่อน
🧠 เทคนิคจำ
"ขวาสุดบวก สลับซ้ายทีละช่อง (ถ้ากำลังคี่)"
🎯 ข้อสอบชอบออก

ออกร่วมกับโจทย์ที่มีตัวประกอบซ้ำ หรือสัมประสิทธิ์นำหน้าติดลบ — ทั้งสองกรณีต้องดึงออกก่อน

☑ Before You Submit
  • ทดสอบค่าตัวเลขในช่วงคำตอบ
  • จุดวิกฤตที่ = ต้องรวม (ทึบ) หรือไม่รวม (โปร่ง) ตามโจทย์

PART 7 · Higher-Degree Inequalities · 🟣 Exam Tip
ตัวประกอบซ้ำ & สัมประสิทธิ์ลบ
🎯
กำลังคู่ซ้ำ → ไม่สลับเครื่องหมาย! / นำหน้าลบ → ดึงลบออกกลับทิศก่อน
👀 เมื่อเจอโจทย์แบบนี้
  • \( (x-2)^2(x+1) > 0 \)
  • \( -x^2+4x-3 > 0 \)
✅ วิธีคิด
  1. กำลังคู่: \( (x-a)^{2k} \ge 0 \) เสมอ → ไม่สลับเครื่องหมายที่จุด \( x=a \) (แตะแล้วกลับ)
  2. กำลังคี่: \( (x-a)^{2k+1} \) → สลับเครื่องหมายปกติ
  3. นำหน้าลบ: คูณ -1 ทั้งสองข้าง กลับทิศอสมการ แล้วค่อยทำตาม step ปกติ
❌ จุดที่พลาดบ่อย
  • สลับเครื่องหมายที่จุดกำลังคู่ทั้งที่ไม่ควรสลับ
  • ลืมกลับทิศอสมการตอนคูณ -1 เพื่อทำให้สัมประสิทธิ์นำเป็นบวก
🧠 เทคนิคจำ
"คี่สลับ / คู่ไม่สลับ (แตะแล้วกลับเส้นเดิม)"
🎯 ข้อสอบชอบออก

ชอบออก \( (x-a)^2(x-b)(x-c) \ge 0 \) ที่มีทั้งกำลังคู่และคี่ปน — ต้องรู้ว่าจุดไหนสลับ จุดไหนไม่สลับ

☑ Before You Submit
  • ระบุกำลังของแต่ละตัวประกอบก่อน แล้วค่อยสลับ/ไม่สลับ
  • ทดสอบค่าตัวเลขในแต่ละช่วงเพื่อยืนยัน
