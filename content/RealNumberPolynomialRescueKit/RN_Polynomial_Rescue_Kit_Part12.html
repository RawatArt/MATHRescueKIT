Top Common Mistakes เรื่องจำนวนจริง
🎯
ไล่เช็ก 7 จุดนี้ก่อนส่งคำตอบ — เสียคะแนนฟรีทุกปี
👀 7 จุดตายที่เด็กเสียคะแนนมากที่สุด
  • คูณไขว้ตัวส่วนในอสมการ (ห้ามเด็ดขาด!)
  • ลืมกลับทิศตอนคูณ/หารด้วยลบ
  • จุดที่ส่วน = 0 ใส่วงกลมทึบแทนที่จะโปร่ง
  • กำลังคู่ซ้ำสลับเครื่องหมาย (ไม่ควรสลับ!)
  • |f| = g(x) ไม่เช็กว่า g(x) ≥ 0
  • อสมการรูท ลืมเงื่อนไขในรูท ≥ 0
  • Vieta ลืมหารด้วย a หรือลืมเครื่องหมายลบ
✅ แนวทางป้องกัน
  1. ทำเสร็จแล้วไล่เช็กลิสต์นี้ก่อนส่งทุกครั้ง
  2. ทดสอบค่าตัวเลขในช่วงคำตอบเสมอ
  3. เขียน Domain ไว้ก่อนเริ่มทุกข้อที่มีตัวส่วน/รูท/log
🧠 เทคนิคจำ
"ห้ามคูณไขว้ / กลับทิศตอนลบ / โปร่งที่ส่วน / คู่ไม่สลับ"
🎯 สถิติจริง

จากข้อสอบ A-Level ที่ผ่านมา: ≥50% ของคะแนนที่เสียในหมวดอสมการมาจาก "ลืมโปร่ง/ทึบ" และ "คูณไขว้"

PART 12 · Ultimate Exam Rescue · 🔵 Concept
Decision Tree: เจอโจทย์แบบไหน → ใช้วิธีไหน
🎯
อ่านโจทย์ → จำแนก pattern → เลือกวิธีที่เร็วที่สุด
👀 ผัง Pattern → วิธี
  • มีค่าสัมบูรณ์ |f| < a → สูตรลัด AND
  • มีค่าสัมบูรณ์ |f| > a → สูตรลัด OR
  • |f| < |g| → ยกกำลังสอง + DoS
  • เศษส่วน > 0 → ย้ายข้าง + sign chart
  • รูท ≤ g → เงื่อนไข 2 ตัว + ยกกำลังสอง
✅ ขั้นตอนการเลือกวิธี
  1. มีเศษส่วน? → ย้ายข้างให้ขวา = 0 ก่อน
  2. มีรูท? → เช็กเงื่อนไข domain ก่อน
  3. มีค่าสัมบูรณ์? → ใช้สูตรลัดถ้า a เป็นค่าคงที่
  4. |f| กับ |g|? → ยกกำลังสองทั้งสองข้าง
❌ จุดที่พลาดบ่อย
  • เลือกวิธีผิดตั้งแต่ต้น ทำให้เสียเวลามากโดยใช่เหตุ
  • ข้ามขั้น "เช็ก domain" ก่อนเริ่มแก้
🧠 เทคนิคจำ
"อ่านรูปแบบ → จับคู่กับ pattern ที่รู้ → ใช้เทคนิคที่เร็วที่สุด"
🎯 ข้อสอบชอบออก

A-Level มักออกโจทย์ผสมที่ต้องใช้หลายเทคนิคพร้อมกัน — แบ่งโจทย์ออกเป็นส่วนๆ ก่อนเลือกวิธี

☑ Before You Submit
  • เช็กว่าคำตอบอยู่ใน domain ที่ถูกต้อง
  • ทดสอบค่าตัวเลขในช่วงคำตอบ

PART 12 · Ultimate Exam Rescue · 🟢 Formula
สูตรลัดรวมฮิต 1 นาทีก่อนสอบ
🎯
ท่องสูตรเหล่านี้ขึ้นใจ — ทุกข้อใช้อย่างน้อยหนึ่งตัว
✅ สูตรที่ต้องจำ
  1. \( a^2-b^2=(a-b)(a+b) \)
  2. \( a^3\pm b^3=(a\pm b)(a^2\mp ab+b^2) \)
  3. \( |f|0)
  4. \( |f|>a \iff f>a \text{ หรือ } f<-a \)
  5. เศษส่วน: ย้ายข้าง ขวา=0 แล้ว sign chart
  6. P(c) = 0 → (x−c) เป็นตัวประกอบ
  7. Vieta: \( \alpha+\beta=-b/a \), \( \alpha\beta=c/a \)
🧠 เทคนิคจำ
"DoS / SOAP / น้อยใน OR มาก / ขวาศูนย์ sign chart / P(c)=0 ตัวประกอบ"
🎯 ข้อสอบชอบออก

ถ้าจำสูตร 7 ตัวนี้ได้ครบ ครอบคลุม ≥80% ของข้อสอบหมวดจำนวนจริงและพหุนาม

☑ Master Checklist
  • Domain เช็กแล้ว (ส่วน≠0 / รูท≥0 / log>0)
  • ทิศอสมการถูก (กลับตอนคูณ/หารลบ)
  • จุดวิกฤตครบ (เศษ=0 AND ส่วน=0)
  • ทดสอบค่าในช่วงคำตอบแล้ว
