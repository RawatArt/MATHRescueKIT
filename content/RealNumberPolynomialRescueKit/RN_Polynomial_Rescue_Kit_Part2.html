PART 2 · Polynomial Divisibility · 🔵 Concept
การหารยาวพหุนาม
🎯
เรียงดีกรีจากมากไปน้อยก่อน ถ้าขาดพจน์ให้เติม 0 คุม
👀 เมื่อเจอโจทย์แบบนี้
  • \( (2x^3 - 3x + 1) \div (x - 2) \)
  • โจทย์ที่ตัวหารมีดีกรีสูงกว่า 1
✅ วิธีคิด
  1. เรียงพหุนามตาม ดีกรีมาก → น้อย เติม 0 ถ้าขาดพจน์ (เช่น \( -3x \) แต่ไม่มี \( x^2 \) ให้เติม \( 0x^2 \))
  2. หารพจน์นำของเศษ ÷ พจน์นำของตัวหาร → ได้พจน์แรกของผลหาร
  3. คูณพจน์นั้นกับตัวหารทั้งก้อน แล้วลบออกจากเศษ
  4. ทำซ้ำจนดีกรีที่เหลือน้อยกว่าตัวหาร
❌ จุดที่พลาดบ่อย
  • ลืมเติม 0 ให้พจน์ที่หายไป ทำให้ column ผิด
  • ลบผิด: \( x^3 - x^3 = 0 \) แต่เขียน \( 2x^3 \) ค้างไว้
🧠 เทคนิคจำ
"เรียงก่อน เติม 0 ถ้าขาด แล้วค่อยหาร"
🎯 ข้อสอบชอบออก

ชอบให้หาเศษจากการหารโดยไม่ต้องหาร ใช้ Remainder Theorem แทนได้เร็วกว่า — รู้จักทั้งสองวิธี

☑ Before You Submit
  • ตรวจ: Dividend = Divisor × Quotient + Remainder
  • ดีกรีของ Remainder ต้องน้อยกว่า Divisor

PART 2 · Polynomial Divisibility · 🟢 Formula
การหารสังเคราะห์ (Synthetic Division)
🎯
ตัวหารต้องเป็น \( x - c \) เท่านั้น แล้วใช้ตัวเลข c แทน
👀 เมื่อเจอโจทย์แบบนี้
  • \( (2x^3 + 3x^2 - x + 5) \div (x - 2) \)
  • ตรวจว่า \( x = c \) เป็นรากของพหุนามหรือไม่
✅ วิธีคิด
  1. เขียนสัมประสิทธิ์ทุกพจน์ตามลำดับ (เติม 0 ถ้าขาด)
  2. วาง c ไว้ซ้าย (ถ้าตัวหารคือ \( x-2 \) → c = 2)
  3. ยกเลขแรกลง คูณกับ c เขียนใต้เลขถัดไป บวกลง ทำซ้ำ
  4. ตัวเลขล่างสุดคือสัมประสิทธิ์ผลหาร ตัวสุดท้ายคือเศษเหลือ
❌ จุดที่พลาดบ่อย
  • ใช้ c = -2 ตอนที่ตัวหารคือ \( x - 2 \) (ต้อง c = +2)
  • ลืมเติม 0 สำหรับพจน์ที่หายไป ทำให้สัมประสิทธิ์เลื่อนผิด
🧠 เทคนิคจำ
"หารด้วย \( x-c \) ใช้ \( +c \) / หารด้วย \( x+c \) ใช้ \( -c \)"
🎯 ข้อสอบชอบออก

ใช้ synthetic division ต่อเนื่อง 2 รอบเพื่อหารพหุนามดีกรี 3 ด้วยตัวประกอบสองตัว

☑ Before You Submit
  • เช็กว่า c ถูกเครื่องหมายหรือไม่
  • ดีกรีผลหาร = ดีกรีเดิม - 1

PART 2 · Polynomial Divisibility · 🟠 Trigger
สมการการหาร (Division Algorithm)
🎯
P(x) = D(x)·Q(x) + R(x) — จำสูตรนี้ไว้ตรวจคำตอบทุกครั้ง
👀 เมื่อเจอโจทย์แบบนี้
  • หาพหุนาม P(x) เมื่อรู้ตัวหาร ผลหาร และเศษ
  • พิสูจน์ว่าผลหารที่ทำถูกต้อง
✅ วิธีคิด
  1. \( P(x) = D(x)\cdot Q(x) + R(x) \)
  2. \( \deg R < \deg D \) เสมอ
  3. ตรวจคำตอบ: คูณ D(x)·Q(x) บวก R(x) แล้วเทียบกับ P(x)
❌ จุดที่พลาดบ่อย
  • คิดว่าเศษต้องเป็นศูนย์เสมอ (ไม่จริง! เศษมีได้)
  • ลืมว่า \( \deg R < \deg D \) ทำให้หารไม่สุด
🧠 เทคนิคจำ
"P = DQ + R คือสูตรตรวจ ใช้เสมอก่อนส่งคำตอบ"
🎯 ข้อสอบชอบออก

ออกในรูปกำหนดเศษและผลหาร แล้วถามว่า P(x) คืออะไร — แทนกลับสูตรได้เลย

☑ Before You Submit
  • ตรวจโดยแทนกลับสูตร P = DQ + R
  • ดีกรีของ R น้อยกว่า D แล้ว
