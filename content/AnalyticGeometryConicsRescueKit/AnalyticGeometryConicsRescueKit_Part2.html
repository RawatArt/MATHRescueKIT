สมการเส้นตรง
🎯
มีจุดหนึ่งจุดกับความชันก็เขียนเส้นได้
👀 เมื่อเจอโจทย์แบบนี้
  • กำหนดจุด \((x_1,y_1)\) และ \(m\)
  • โจทย์บอกผ่านจุดนี้และมีความชันนี้
✅ วิธีคิด
  1. เขียนรูป \(y - y_1 = m(x - x_1)\)
  2. แทนค่าจุดและ m
  3. จัดรูปถ้าต้องการ
❌ จุดที่พลาดบ่อย
  • เอา \(x_1,y_1\) ไปสลับตำแหน่ง
  • ใช้สูตรผิดเป็น \(y = mx + c\) ทั้งที่ยังหา c ไม่ได้
🧠 เทคนิคจำ
"จุดหนึ่ง m หนึ่ง ใช้ point-slope"
🎯 ข้อสอบชอบออก

ข้อสอบมักให้ข้อมูลครบแค่ 'ผ่านจุด' กับ 'ความชัน' แล้วดูว่าเขียนสมการได้เร็วไหม ต้องจำรูปนี้ให้ขึ้นใจ

☑ Before You Submit
  • มีจุดแล้วหรือยัง
  • แทนค่าในรูป point-slope ถูกไหม

ระยะจุดถึงเส้น
🎯
เอาจุดยัดในสมการเส้น แล้วหารด้วยราก
👀 เมื่อเจอโจทย์แบบนี้
  • หาจากจุด \((x_1,y_1)\) ถึงเส้น \(Ax + By + C = 0\)
  • โจทย์ถามระยะจากจุดถึงเส้นตรง
✅ วิธีคิด
  1. แทนพิกัดจุดใน \(|Ax_1 + By_1 + C|\)
  2. หาค่า \(\sqrt{A^2 + B^2}\)
  3. เอาค่าสัมบูรณ์หาร
❌ จุดที่พลาดบ่อย
  • ลืมค่าสัมบูรณ์
  • ใช้พิกัดผิดช่อง
🧠 เทคนิคจำ
"แทนจุด หารด้วยรากของสัมประสิทธิ์"
🎯 ข้อสอบชอบออก

ชอบออกแบบมีเลขติดลบเยอะเพื่อหลอกเครื่องหมาย อย่าลืมว่าค่าระยะต้องไม่ติดลบ

☑ Before You Submit
  • ใส่จุดลงสมการแล้วหรือยัง
  • ตัวส่วนเป็น \(\sqrt{A^2+B^2}\) ไหม

PART 2 · LineDistanceEquations · 🟣 Exam Tip
ระยะเส้นคู่ขนาน
🎯
A B ต้องเหมือนก่อน ค่อยลบ C
👀 เมื่อเจอโจทย์แบบนี้
  • หา ระยะระหว่าง \(2x - 3y + 7 = 0\) กับ \(2x - 3y - 5 = 0\)
  • โจทย์ให้เส้นขนาน 2 เส้น
✅ วิธีคิด
  1. เช็กว่า \(A\) และ \(B\) เท่ากันก่อน
  2. ใช้ \(d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}\)
  3. คำนวณแล้วตอบ
❌ จุดที่พลาดบ่อย
  • จับลบทั้งที่ A, B ไม่เท่ากัน
  • ลืมดูว่าเครื่องหมายของ C เป็นอะไร
🧠 เทคนิคจำ
"A B เหมือน ค่อยเอา C ลบ"
🎯 ข้อสอบชอบออก

ข้อสอบมักซ่อนกับดักตรงสมการไม่ได้จัดรูปเหมือนกัน ต้องแปลงให้อยู่รูปมาตรฐานก่อนทุกครั้ง

☑ Before You Submit
  • A, B เท่ากันหรือยัง
  • จัดรูปเป็นมาตรฐานก่อนหรือยัง
