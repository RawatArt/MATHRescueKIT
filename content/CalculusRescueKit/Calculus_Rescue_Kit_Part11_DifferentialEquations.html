PART 11 · Differential Equations · 🔵 Concept
Separation of Variables
🎯
แยก x กับ y ได้แต่ละข้าง → ย้ายไปคนละฝั่งแล้วอินทิเกรตทั้งคู่
👀 เมื่อเจอโจทย์แบบนี้
  • \( \dfrac{dy}{dx}=xy \)
  • \( \dfrac{dy}{dx}=\dfrac{x}{y} \)
✅ วิธีคิด
  1. จัดให้พจน์ y ทั้งหมดอยู่ฝั่ง dy พจน์ x ทั้งหมดอยู่ฝั่ง dx
  2. อินทิเกรตทั้งสองข้างแยกกัน
  3. ใส่ +C เพียงครั้งเดียว (รวมไว้ข้างเดียวก็พอ)
  4. แก้สมการให้อยู่ในรูป y = ... ถ้าโจทย์ต้องการ
❌ จุดที่พลาดบ่อย
  • ใส่ +C ทั้งสองข้าง (ซ้ำซ้อน ที่จริงรวมเป็นค่าเดียวพอ)
  • แยกตัวแปรผิด เหลือ x ปนอยู่ฝั่ง y
🧠 เทคนิคจำ
"y อยู่กับ dy, x อยู่กับ dx แยกให้สุดก่อนอินทิเกรต"
🎯 ข้อสอบชอบออก

เป็นเทคนิคพื้นฐานที่สุดของสมการเชิงอนุพันธ์ มักผสมกับ u-substitution ตอนอินทิเกรต

☑ Before You Submit
  • เช็กว่าแยกตัวแปรสุดแล้วก่อนอินทิเกรต
  • มี +C เพียงครั้งเดียว

PART 11 · Differential Equations · 🟢 Formula
Initial Value Problems
🎯
มีเงื่อนไขเริ่มต้นให้ → แทนค่าหา C เฉพาะเจาะจง อย่าตอบแค่คำตอบทั่วไป
👀 เมื่อเจอโจทย์แบบนี้
  • แก้ \( \dfrac{dy}{dx}=2x \) เมื่อ \( y(0)=3 \)
✅ วิธีคิด
  1. แก้สมการเชิงอนุพันธ์เพื่อหาคำตอบทั่วไป (มี +C)
  2. แทนเงื่อนไขเริ่มต้น \( (x_0,y_0) \) ลงในคำตอบทั่วไป
  3. แก้หาค่า C ที่เฉพาะเจาะจง
  4. เขียนคำตอบเฉพาะ (particular solution) แทน C ด้วยค่าที่หาได้
❌ จุดที่พลาดบ่อย
  • ตอบแค่คำตอบทั่วไปที่มี C ค้างไว้ ไม่แทนค่า C ที่หาได้
  • แทนเงื่อนไขเริ่มต้นผิดตัวแปร (สลับ x กับ y)
🧠 เทคนิคจำ
"หาคำตอบทั่วไปก่อน แทนเงื่อนไขหา C แล้วเขียนคำตอบเฉพาะ"
🎯 ข้อสอบชอบออก

มักผสมกับโจทย์ฟิสิกส์ เช่นหาตำแหน่งจากความเร็วเริ่มต้นและตำแหน่งเริ่มต้นที่กำหนด

☑ Before You Submit
  • เช็กว่าแทนค่า C กลับเข้าคำตอบสุดท้ายแล้ว ไม่เหลือ C ค้าง
